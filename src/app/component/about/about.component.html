Instructions
  • Calculations submitted will appear in the Global Calculations window
  • Enter a valid user name and connect. Valid name cannot be of size less than 4
  • Use = button to calculate the result
  • Click Send to submit your expression to the global calculations
  • clear button clears everything and Del button deletes the last character
  • You can only submit your calculation iff its not empty and valid. An expression is valid if it contains =
  • Once you click =, you can only use Del, Send or Clear buttons
  • May contain bugs
  • Have fun !
